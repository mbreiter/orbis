check_move_to_destination(destination)¶

Checks whether this unit can move one tile on the path to the given Point. This will use the built in path-finding system to find the next move along the shortest path. Does not actually move the unit. This check does not take into account the positions of friendly and enemy units as they might change by the time the move is actually performed.

Parameters:destination ((int,int)) – The point to move towards.
Return type:MoveResult
Returns:The expected result if you were to try to move. If this is NO_MOVE_ATTEMPTED, then a path could not be found to the destination point.

move(direction)¶

Orders this Unit to move one tile in the given direction at the end of this turn. Note that the unit will not move until the turn is over.

Parameters:direction (Direction) – The direction to move in
Returns:The result of attempting to make this move. If this is MOVE_VALID, this does not guarantee the move will succeed. The move will succeed ONLY if no enemy or friendly stays in, or attempts to move to the tile in the given Direction. Thus you won’t know the final result of the move until the current turn completes. You can find out the result of the move from the previous turn from FriendlyUnit.getLastMoveResult().
Return type:MoveResult

get_next_direction_in_path(start, destination)¶

Gets the next direction in the shortest path from the given start tile to the given destination. Avoids walls, but not units. This function will run extremely quickly if the map has a navigation cache (.nac) file. All maps provided in the game and tournament will have this file. Check the manual for instructions on generating the files for your own custom maps.

Parameters:
  • start ((int,int)) – The tile to start path finding from
  • destination ((int,int)) – The tile to try to find a path to
Return type:

Direction
Returns:

The direction to move in from the start tile to get to the destination tile. Returns NOWHERE if there is no way to get to the destination or if the start tile is the same as the destination (you are already there).
